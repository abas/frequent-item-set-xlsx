# Apriori Algorithm
searching frequent item set on xlsx data set

## System Requirement
- nodejs & npm


## Installation Instruction
``` bash
{clone project}
cd {project}
npm install
```

## Data Set Directory
``` dir
/doc/{filename}
```

## Running Program
``` javascript
node apriori.js
```
